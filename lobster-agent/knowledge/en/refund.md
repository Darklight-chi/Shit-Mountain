# Return & Refund Policy

## Standard Returns
- Returns accepted within 7 days of receipt
- Items must be in original packaging and unused
- Return shipping is the buyer's responsibility

## Defective Items
- Defective items qualify for free return shipping
- Please provide photos of the defect

## Refund Process
1. Submit return request
2. Seller approves
3. Ship item back
4. Seller confirms receipt
5. Refund issued (3-5 business days)

## Non-returnable Items
- Past 7-day window
- Used or damaged items
- Custom/personalized items
